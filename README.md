# Dataset: Road Damage Dataset

Este dataset contiene imágenes de daños viales (baches, grietas, alcantarillas).



**Fuente original:** [Kaggle - Road Damage Dataset](https://www.kaggle.com/datasets/lorenzoarcioni/road-damage-dataset-potholes-cracks-and-manholes)



**Cómo descargarlo:**

No es necesario descargar las imágenes manualmente. Al ejecutar el notebook `dev/01_dataset_preparation.ipynb`, el dataset se descargará y organizará automáticamente en esta carpeta utilizando la librería `kagglehub`.
